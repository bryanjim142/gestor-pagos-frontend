# 💳 Sistema de Gestión de Nómina y Pagos — Frontend

> Aplicación web para gestión de empleados, nómina y pagos.

![React](https://img.shields.io/badge/React-18-61DAFB?logo=react)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-F7DF1E?logo=javascript)
![Vite](https://img.shields.io/badge/Vite-5-646CFF?logo=vite)

## ¿Qué hace este proyecto?

- 👤 Registro y gestión de empleados
- 💰 Registro de nómina por empleado
- 💳 Gestión de pagos por persona
- 🔐 Cuenta individual para cada empleado

## Tecnologías

- React + Vite
- JavaScript

## Instalación

git clone https://github.com/bryanjim142/Gestor-Pagos-Frontend
cd Gestor-Pagos-Frontend
npm install
npm run dev
